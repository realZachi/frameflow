# Support

Frameflow is maintained by the community on a best-effort basis.

## Get help

Stuck? Open a GitHub issue and describe what you're running into. Browser, operating system, and the steps that led there make it easier to help, but don't let a missing detail stop you from asking.

The [README](README.md) covers API-key setup and self-hosting, which is where most setup questions land.

Please leave out API keys, private screenshots, customer data, and anything else sensitive when posting logs or images.

## Where things belong

| Topic | Channel |
| --- | --- |
| Something is broken | Bug report |
| An idea or improvement | Feature request |
| Setup or usage question | GitHub issue |
| Security vulnerability | Private process in `SECURITY.md` |
| Code of Conduct incident | Private process in `CODE_OF_CONDUCT.md` |

Vulnerabilities and personal disputes are the two things that shouldn't go in a public issue.

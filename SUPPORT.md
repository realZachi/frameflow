# Support

Frameflow is maintained by the community on a best-effort basis.

## Get help

Before asking for help:

1. Read the [README](README.md), especially API-key setup and self-hosting.
2. Search existing GitHub issues.
3. Run `bun run check` if the problem occurs in a local development copy.

If the problem still exists, open a GitHub issue with the appropriate form. Include the browser, operating system, commit or version, reproduction steps, expected behavior, and actual behavior.

Please remove API keys, private screenshots, customer data, and other sensitive information before posting logs or images.

## Where each report belongs

| Topic | Channel |
| --- | --- |
| Reproducible defect | Bug report |
| Product idea or enhancement | Feature request |
| Setup or usage question | GitHub issue |
| Security vulnerability | Private process in `SECURITY.md` |
| Code of Conduct incident | Private process in `CODE_OF_CONDUCT.md` |

Public issues are not a suitable place for vulnerabilities or personal disputes.
